DeerFoliaPlus 是基于 [DeerFolia](/core/DeerFolia/) 的增强版本。相比于 DeerFolia，DeerFoliaPlus 引入了更多额外功能，这些功能并非性能优化方面的改动，而是增强了服务器的功能性。

DeerFoliaPlus 继承了 DeerFolia 的所有特性（刷沙机制、生物动态激活、异步寻路、实体节流、网络优化、POI 优化等），并在此基础上添加了以下功能。

## 此分支特性

### FakePlayer 系统

假人系统 (Leaves/Lumina) — 支持创建可控制的假人玩家，用于测试、刷怪塔自动化等场景。

### 基岩版风格要塞生成

Bedrock-style Stronghold Generation — 随机无限分布代替 Java 版的 128 个环状分布。

### 协议支持

- **Syncmatica Protocol** — 服务端与客户端之间共享 Litematica 投影，支持上传/下载/修改
- **Servux Protocol** — 为 MiniHUD/Litematica 提供结构边界框叠加层和实体 NBT 数据查看
- **Recipe Sync Protocol** — 向 NeoForge/Fabric 客户端同步完整配方数据，支持 JEI 等模组显示服务器配方

### 自定义配方系统

Custom Recipe System — 通过配置文件定义自定义合成、烧炼、营火烹饪、高炉、切石机等配方。

### 实用命令

- 快速设置时间/天气（`/day`、`/noon`、`/night`、`/rain`、`/storm`、`/sun`）
- 飞行模式切换（`/fly`）
- 无敌模式切换（`/god`）
- 快速打开末影箱和各类工作台（`/enderchest`、`/workbench`、`/craftingtable`、`/stonecutter`、`/loom`、`/grindstone`、`/anvil`、`/cartographytable`、`/smithingtable`）
- 物品相关（`/hat`、`/more`、`/suicide`）
- 玩家背包查看（`/inspect`）
