DeerFoliaPlus 是基于 [DeerFolia](/core/DeerFolia/) 的增强版本。相比于 DeerFolia，DeerFoliaPlus 引入了更多额外功能，这些功能并非性能优化方面的改动，而是增强了服务器的功能性。

DeerFoliaPlus 继承了 DeerFolia 的所有特性（刷沙机制、生物动态激活、异步寻路、实体节流、网络优化、POI 优化等），并在此基础上添加了以下功能。

## 此分支特性

- 假人系统：FakePlayer (Leaves/Lumina)
- 基岩版风格要塞生成：Bedrock-style Stronghold Generation
