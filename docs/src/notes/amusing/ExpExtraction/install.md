## 配置参考

```yaml
# 使用 /extract 命令生成经验瓶
command-extraction:
  enable: true
  # 每个附魔瓶可容纳的最大经验值
  maximum: 2000
# 使用空瓶 Shift+右键 点击附魔台生成经验瓶
enchanting-table:
  enable: true
  amount: 100
```

## 权限节点

| 权限节点                  | 说明   | 默认   |
|-----------------------|------|------|
| expextraction.extract | 提取经验 | true |
